# Melate Retro - Análisis Estadístico

Este archivo contiene el análisis estadístico automatizado para el sorteo Melate Retro, siguiendo la misma metodología aplicada a Melate, Revancha y Revanchita.

## Instrucciones de uso

1. Descarga el archivo histórico de Melate Retro desde:
   https://www.loterianacional.gob.mx/Home/Historicos?ARHP=TQBlAGwAYQB0AGUALQBSAGUAdAByAG8A
   (guárdalo como `MelateRetro.csv` en la raíz del proyecto)
2. Ejecuta el script `melate_retro.py` para generar el análisis y visualizaciones.
3. El resultado se guardará en `ANALISIS_RETRO.md` y gráficos PNG.

## Archivos generados
- `MelateRetro.csv`: Datos históricos de Melate Retro
- `melate_retro.py`: Script de análisis y generación de visualizaciones
- `ANALISIS_RETRO.md`: Reporte en markdown con resultados y recomendaciones
- `retro_*.png`: Gráficos de frecuencias y calor

## Metodología
- Análisis de frecuencias absolutas y desviaciones
- Clasificación de números por calor (muy caliente, caliente, normal, frío, muy frío)
- Visualización de resultados
- Recomendaciones de estrategia basadas en los datos
